# 🖥️ IT Translator - Переводчик IT-терминов

## Описание

**IT Translator** — это интерактивное приложение на Python для перевода компьютерных терминов и слов с русского на английский язык. Идеально подходит для студентов, программистов и специалистов IT-индустрии, которые хотят расширить свой технический словарный запас.

## Что такое информатика?

Информатика — это наука об информации и её обработке с помощью компьютеров. Она изучает алгоритмы (последовательность действий для решения задачи), языки программирования, структуры данных, операционные системы и сетевые технологии. Современная информатика охватывает множество областей: разработка программного обеспечения, кибербезопасность, облачные вычисления, искусственный интеллект и машинное обучение. Профессионалы в этой сфере используют специализированные термины на английском языке, поэтому знание IT-словаря критически важно. Данный переводчик содержит более 100 основных терминов, необходимых каждому, кто работает в области информационных технологий.

## Особенности

✨ **Перевод слов** — быстрый перевод IT-терминов с русского на английский  
🔍 **Обратный поиск** — поиск русского слова по английскому переводу  
📚 **Полный словарь** — просмотр всех 100+ терминов в базе  
➕ **Добавление слов** — расширение словаря собственными терминами  
🗑️ **Удаление слов** — управление содержимым словаря  
🎯 **Интерактивное меню** — удобный интерфейс для работы  

## Структура кода

- **`ITTranslator`** — основной класс с методами для перевода
- **`translate(word)`** — переводит одно слово
- **`translate_text(text)`** — переводит несколько слов
- **`search_by_english(word)`** — обратный поиск
- **`get_dictionary()`** — получить полный словарь
- **`show_all_words()`** — вывести все слова
- **`add_word(rus, eng)`** — добавить новое слово
- **`remove_word(rus)`** — удалить слово
- **`main()`** — интерактивное меню

### Примеры использования

**Перевод одного слова:**
```
👉 Выберите опцию (1-7): 1
Введите слово на русском: программа
✨ Перевод: program
```

**Перевод нескольких слов:**
```
👉 Выберите опцию (1-7): 2
Введите текст (слова через пробел): компьютер программа код
✨ Перевод: computer program code
```

**Обратный поиск:**
```
👉 Выберите опцию (1-7): 3
Введите слово на английском: database
✨ Русское слово: база данных
```

## Содержимое словаря

Переводчик включает основные IT-термины:
- Общие термины: программа, код, данные, алгоритм
- Аппаратное обеспечение: компьютер, процессор, память
- Сети: сеть, протокол, сервер, клиент, интернет
- Базы данных: база данных, запрос, таблица, индекс
- ОС и системное ПО: операционная система, драйвер
- Разработка: функция, класс, переменная, объект
- Безопасность: кибербезопасность, шифрование, firewall
- Современные технологии: облако, AI, machine learning
